# Security Policy

本專案是本機執行的 Windows 桌面工具，不需要網路連線，也不應收集或上傳使用者資料。

## 回報安全問題

如果你發現安全問題，請優先使用 GitHub 的私密漏洞回報或 Security Advisory 功能聯絡維護者。請不要在公開 issue 中貼出可被濫用的細節、個人診斷輸出、硬體唯一識別資訊或原始封包紀錄。

## 使用者資料

請不要提交以下資料到公開 repository：

- `debug_packets.txt`
- `detected_tablets.txt`
- `build/` 或 `dist/` 產物
- 含本機路徑、硬體唯一 ID、token、金鑰或帳號資料的診斷輸出
